# SISTEMA DE GESTIÓN DE INVENTARIO - GRUPO 3

## Descripción del proyecto
Aplicación CLI con Python, que administra productos y proveedores, registrando cada movimiendo hecho de inventario (entradas y salidas) y actualizando automáticamente las cantidades disponibles con validaciones para evitar stock negativo
